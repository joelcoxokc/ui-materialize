HTML Setup

In Order for ui-materialize to work properly, we must layout our initail HTML
The best way to do this is using either ng-include, or multiple views from the first page.


        <html>
            <body class="nav-fixed">

                <mz-nav color="white" fixed=true>
                    <div ng-include="'path/to/nav.template.html'"></div>
                </mz-nav>

                <mz-view></mz-view>

                <mz-footer color="grey darken-3">
                    <div ng-include="'path/to/footer.template.html'"></div>
                </mz-footer>

            </body>
        </html>
